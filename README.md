# Portfolio Website

A modern, responsive portfolio website built with **React**.  
This project showcases my skills, projects, and services, with smooth navigation and a clean UI.

---

## 🚀 Features

- **Hero Section** – Introduction and personal branding
- **Projects Section** – Highlighted work with descriptions
- **Services Section** – Overview of offered services
- **Footer** – Contact details and social links
- **Back to Top Button** – Smooth scroll for better UX
- **Responsive Design** – Optimized for desktop and mobile

---

## 🛠️ Tech Stack

- **React** (functional components, hooks)
- **JavaScript (ES6+)**
- **HTML5 & CSS3**
- **Git & GitHub** (version control and hosting)

## ⚙️ Installation & Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/Momna533/portfolio-v1
   ```

````

2. Navigate to the project folder:
   ```bash
   cd portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open in browser at:
   ```
   http://localhost:3000
   ```
````
